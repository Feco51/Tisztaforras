# Tisztaforras
Ez a projekt egy weboldal fejlesztéséről és teszteléséről szól, ami a Tiszta Forrás névre hallgat.

# Megnyitása: 
1.Töltsük le a .zip kiterjesztésű fájlt.
2.Csomagoljuk ki egy kicsomagoló programmal. (pl. 7zip)
3.Nyissuk meg a kicsomagolt mappát.
4.Keressük meg benne az "index.html" fájlt.
5.Dupla kattintásal nyisuk meg.
+ Ha az operációs rendszer megkérdezi, hogy milyen programmal nyissa ki, válasszon valamilyen böngészőt. 

# Szerkesztése:
1.Töltsük le a .zip kiterjesztésű fájlt.
2.Csomagoljuk ki egy kicsomagoló programmal. (pl. 7zip)
3.Nyissuk meg a szerkesztőprogramot.
4.Importáljuk be a kicsomagolt mappát.
5.Mehet a szerkesztés :)
